# Customer-Behavior-Analysis_End-to-End-Data-Analytics-Project
Customer Behavior Analysis End-to-End Data Analytics Project. 

This project demonstrates a complete end-to-end data analytics workflow — from raw data processing to interactive dashboard visualization.
The solution integrates:
1.Python (Jupyter Notebook) for data cleaning and analysis
2.PostgreSQL for structured data storage and querying
3.Power BI for interactive dashboard creation

Project Architecture:
Raw Data → Python Cleaning → PostgreSQL Database → Power BI Dashboard

Key Objectives:

1.Analyze customer purchase behavior
2.Identify high-performing product categories
3.Understand customer segmentation and subscription trends
4.Discover age-group based purchasing patterns

Dashboard Features:

1.KPI cards for customer count, average rating, and purchase amount
2.Category-wise revenue and sales comparison
3.Customer distribution by subscription status
4.Purchase trends across different age groups
5.Shipping preference insights

Key Insights

1.Clothing category generates the highest revenue.
2.A large portion of customers are non-subscribers.
3.Young Adult and Middle-Age groups drive most of the purchases.

How to Run This Project:

Run the Jupyter notebook to clean and preprocess the data
              👇
Load processed data into PostgreSQL
               👇
Execute SQL queries for analysis
                👇
Open the Power BI file to explore the dashboard
